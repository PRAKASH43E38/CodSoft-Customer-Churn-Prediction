# CodSoft Customer Churn Prediction

A Machine Learning project developed as part of the **CodSoft Machine Learning Internship** to predict whether a customer is likely to leave a telecom service based on customer attributes and usage patterns.

## Project Overview

Customer churn prediction helps businesses identify customers who are likely to discontinue their services. This project uses machine learning techniques to analyze customer data, preprocess features, train classification models, and evaluate their performance.

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Customer Churn Prediction
